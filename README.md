
### Prerequisites
- linux
- [clementine](https://www.clementine-player.org/)
- nodejs
- npm

### Installation
1. download this project
2. cd to this repository in your local machine
3. install dependencies by executing
```sh
  npm install
```
4. execute
```
node music.js
```
5. point your browser to http://localhost:8080/
